Си

Мощный язык программирования высокого уровня. Используется для системного, прикладного и встроенного программирования, обеспечивает низкоуровневый доступ к ресурсам компьютера.

C++

Высокоуровневый, многозадачный, объектно-ориентированный язык программирования. Используется для разработки приложений, игр, системного программирования и создания эффективных структур данных.

C#

Мощный, объектно-ориентированный язык программирования Microsoft. Используется для создания Windows-приложений, веб-приложений и игр с использованием платформы .NET.

Java

Универсальный, кроссплатформенный язык программирования. Используется для создания приложений, веб-серверов, мобильных приложений (Android), игр и других программ.

Spring

Мощный фреймворк для Java, облегчающий создание и управление приложениями, обеспечивая высокую производительность, безопасность и эффективность разработки.

MySQL

Это система управления базами данных, которая используется для хранения, организации и обработки данных в различных приложениях и веб-сервисах.
